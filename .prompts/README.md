# Task-Specific Prompts

Quick-reference prompts for specific tasks. Use with `#file` in Copilot Chat.

> **Note:** Base project context and constraints are automatically included from [.github/copilot-instructions.md](../.github/copilot-instructions.md).

## Usage

In Copilot Chat, reference a prompt file:
```
#file:.prompts/dotnet-build.md [your task description]
```

## Available Prompts

| File | Use For |
|------|---------|
| [dotnet-build.md](dotnet-build.md) | Build optimization, publish config, warnings |
| [dotnet-code.md](dotnet-code.md) | Features, refactoring, bug fixes |
| [docker.md](docker.md) | Dockerfile changes, image optimization |
| [github-actions.md](github-actions.md) | CI/CD workflow changes |
| [lua-scripting.md](lua-scripting.md) | DCS World Lua scripts |
